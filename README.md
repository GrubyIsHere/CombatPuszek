# CombatPuszek
Fancy combat-type robot with huge saw on top.

![Angled version](angled.png)

![Flattop version](flat.png)

## Features
*   ESP32-based remote control system.
*   All-around horizontal saw, up to 250mm in diameter.
*   3lbs (1.36kg) weight category.
*   Two drive configurations: wheels or soft tracks.
*   Swappable motor drivers, main ESC, and even main ESP32 module. 
*   Powered by 3S LiPO.
